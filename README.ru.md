Современный, быстрый и многофункциональный загрузчик медиафайлов, созданный с использованием Tauri v2, Vue 3 и Rust.

## От Python к Tauri + Rust

> Оригинальная версия на PyQt6 находится по ссылке: [Magerko/universal-media-downloader](https://github.com/Magerko/universal-media-downloader) (Оригинальный автор: @Magerko)

Исторически это приложение было написано на Python с использованием PyQt6. Хотя оно работало, оно имело ограничения в плане визуального оформления (UI), производительности и размера дистрибутива.

**Почему Tauri?**
1. **Производительность**: Бэкенд на Rust выполняет операции невероятно быстро и потребляет очень мало оперативной памяти по сравнению со старым демоном на Python.
2. **Нативный UX и эстетика**: Переход на Vue 3 на фронтенде позволил создать полностью кастомизируемый, красивый и плавный интерфейс с системными CSS-переменными, поддержкой тем (Светлая/Тёмная), эффектом стекла (Glassmorphism) и аппаратно ускоренными анимациями, чего было крайне трудно достичь в PyQt6.
3. **Кроссплатформенность**: Tauri компилирует приложение в единый компактный нативный исполняемый файл (exe/deb/appos) без необходимости установки громоздкого окружения Python.

### Ключевые улучшения
- ✨ **Улучшенный интерфейс (UI/UX)**: Полностью переработанный дизайн, современные тени, плавные CSS-переходы и микро-анимации.
- ⚡ **Оптимизированная очередь загрузок**: Асинхронный менеджер загрузок на Rust обеспечивает надежное параллельное скачивание.
- 🍪 **Извлечение Cookies**: Встроенная поддержка локального извлечения cookies прямо из вашего браузера (полезно для закрытых видео/стримов).
- 🔄 **Отслеживание в реальном времени**: Реактивные прогресс-бары для отслеживания упаковки `.mp4` и обработки `FFmpeg` в прямом эфире без блокировки UI.

## Возможности

- **Мультиплатформенность**: Скачивайте видео и аудио с YouTube, TikTok, Instagram, VK, RuTube, Twitch, Kick, X (Twitter), Facebook и многих других сайтов (на базе мощного `yt-dlp`).
- **Параллельная загрузка**: Скачивайте несколько файлов одновременно, а не по одному.
- **Авто-объединение**: Использование локального FFmpeg для автоматической склейки видео и аудио потоков максимального качества.
- **Настройка качества**: Тонкая настройка выбора 4K, 1080p, 720p или режимов "Только аудио (mp3)".
- **История**: Встроенная локальная база данных для хранения истории загрузок с возможностью быстрого повторного скачивания или открытия файла директорией.

## Установка и запуск

### Требования

Для запуска или сборки этого проекта (разработки) вам понадобятся следующие инструменты:
- [Node.js](https://nodejs.org/) (v16 или выше)
- [Rust](https://www.rust-lang.org/tools/install)
- [FFmpeg](https://ffmpeg.org/download.html) (Должен быть в системном PATH или в папке `assets/ffmpeg/bin`)
- [yt-dlp](https://github.com/yt-dlp/yt-dlp) (или может быть установлен/обновлен через само приложение)
- [Deno](https://deno.com/) (необходим для обхода блокировок и сложных скриптов обфускации YouTube)

### Разработка

Запуск сервера разработки с горячей перезагрузкой (Hot-Reload):

```bash
cd tauri-app
npm install
npm run tauri dev
```

### Сборка для релиза

Чтобы создать готовый нативный установщик (DEB, MSI, AppImage и т.д.) для вашей ОС, выполните в корне:

```bash
cd tauri-app
npm run tauri build
```

После завершения сборки готовый инсталлятор будет находиться в директории `tauri-app/src-tauri/target/release/bundle/`.
